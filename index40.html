Tutorial Dasar JavaScript

Apa itu JavaScript?

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web lebih interaktif dan dinamis.

Contoh Dasar JavaScript

Berikut adalah contoh skrip JavaScript sederhana untuk menampilkan teks di halaman web:

            <script>
                document.write("Hello, world!");
            </script>

Variabel dalam JavaScript

Di JavaScript, variabel dapat dideklarasikan menggunakan var, let, atau const:

            <script>
                let nama = "Budi";
                console.log("Halo, " + nama);
            </script>

Kondisional dalam JavaScript

JavaScript mendukung pernyataan kondisional seperti if dan else:

            <script>
                let angka = 10;
                if (angka > 5) {
                    console.log("Angka lebih besar dari 5");
                } else {
                    console.log("Angka 5 atau lebih kecil");
                }
            </script>

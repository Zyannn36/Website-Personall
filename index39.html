Tutorial Dasar PHP

Apa itu PHP?

PHP (Hypertext Preprocessor) adalah bahasa pemrograman server-side yang digunakan untuk mengembangkan aplikasi web dinamis.

Contoh Dasar PHP

Berikut adalah contoh skrip PHP sederhana untuk menampilkan teks:

            <?php
                echo "Hello, world!";
            ?>

Variabel dalam PHP

Di PHP, variabel diawali dengan tanda $. Contoh penggunaan variabel:

            <?php
                $nama = "Budi";
                echo "Halo, " . $nama;
            ?>

Kondisional dalam PHP

PHP mendukung pernyataan kondisional seperti if dan else:

            <?php
                $angka = 10;
                if ($angka > 5) {
                    echo "Angka lebih besar dari 5";
                } else {
                    echo "Angka 5 atau lebih kecil";
                }
            ?>

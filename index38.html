Tutorial Dasar CSS

Apa itu CSS?

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mendesain tampilan halaman web. CSS memungkinkan Anda untuk mengubah warna, ukuran, font, dan tata letak elemen di halaman.

Contoh Dasar CSS

Berikut adalah contoh bagaimana CSS digunakan untuk memberi gaya pada elemen HTML:

            <style>
                body {
                    background-color: lightgray;
                    font-family: Arial, sans-serif;
                }
                h1 {
                    color: blue;
                }
                p {
                    font-size: 16px;
                }
            </style>

Selektor dalam CSS

CSS menggunakan berbagai selektor untuk memilih elemen HTML yang akan diberi gaya:

  • Selektor Elemen: h1 { color: red; }
  • Selektor ID: #header { font-size: 24px; }
  • Selektor Kelas: .judul { font-weight: bold; }
